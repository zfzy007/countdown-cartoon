# AGENTS.md

## Mission
Provide minimal, clear, and deterministic instructions for agents contributing to this project.

## Commands
- **Install dependencies**: This project doesn't use package managers. Ensure `index.html`, `script.js`, and `style.css` are present in the workspace.
- **Start development server (optional)**: Use any local HTTP servers to host `index.html` (e.g., `python -m http.server`).
- **Lint/Coding Standards**: Ensure script formatting aligns with basic browser compatibility.
